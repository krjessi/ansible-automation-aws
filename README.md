# Ansible Automation Project on AWS

## Overview

This project demonstrates Infrastructure Automation and Configuration Management using Ansible on AWS EC2 instances.

## Infrastructure

- 1 Ansible Master Node
- 2 Target Servers
- AWS EC2 Environment

## Features

- Passwordless SSH Authentication
- Inventory Management
- Ad-Hoc Commands
- Playbooks
- Roles
- Templates
- Handlers
- Group Variables
- Host Variables
- Ansible Vault
- Java 21 Installation
- Nginx Deployment
- Spring Boot Deployment Simulation
- Systemd Service Management

## Technologies

- AWS EC2
- Ubuntu Linux
- Ansible
- Java 21
- Nginx
- Systemd
